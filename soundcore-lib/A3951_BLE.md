# Soundcore Liberty Air 2 Pros

Soundcore Liberty Air 2 Pros (A3951) may advertise with a different MAC Address and the regular MAC address from a Classic Bluetooth scan will not have any BLE Data(Services, Charactiristics, etc).

The BLE Advertisement packet contains the regular advertised MAC so we can replace that MAC in the results
with the BLE one, so we can retrieve Services etc.

All this is needed because we rely on the Bluetooth Name to detect the type of the device. So if another way is found, that operation is not needed. 

## Note
The packet `[1, 1A, F5, DA]` may be a device identification packet. Some investigation is needed.

## Example of Advertisement data
```
BLE SCAN MAC: "AC:12:2F:6B:B3:DC"
Advertisement data(hex): [6]
Advertisement data(hex): [AC, 12, 2F, 6A, D2, 7, 0, 0]
Advertisement data(hex): [1, 1A, F5, DA]
```
